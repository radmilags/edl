#TAD Lista

O TAD Lista modela um sequência de posições armazenando objetos quaisquer

Ele estabelece uma
relação antes/depois
entre posções

##Métodos genéricos
*n size(), isEmpty()

##Métodos de fila:
n isFirst(n), isLast(n)

##Métodos para acessar:
n first(), last()
n before(p), after(p)

##Métodos para atualizar:
n replaceElement(n, o),
swapElements(n, q)
n insertBefore(n, o),
insertAfter(n, o),
n insertFirst(o),
insertLast(o)
n remove(n)
